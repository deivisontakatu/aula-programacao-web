# 🔌  Acesso a Banco de Dados

O acesso ao banco de dados é realizado por meio de linguagens de programação e APIs específicas.

---

## ⚙️ Etapas do Acesso

1. Abrir conexão com o banco
2. Enviar comandos (SQL ou equivalente)
3. Processar resultados
4. Fechar conexão

---

## 💻 Formas de Acesso

### 📌 Acesso Direto (SQL)

* Uso de comandos SQL explícitos
* Maior controle

**Exemplo:**

```sql
SELECT * FROM usuarios;
```

---

### 📌 Acesso via API/Driver

* JDBC (Java)
* PDO (PHP)
* psycopg2 (Python)

---

## 🧪 Operações Básicas (CRUD)

| Operação | Descrição       |
| -------- | --------------- |
| Create   | Inserir dados   |
| Read     | Consultar dados |
| Update   | Atualizar dados |
| Delete   | Remover dados   |

---

## ⚠️ Boas Práticas

* Usar **prepared statements** (evitar SQL Injection)
* Gerenciar conexões (pooling)
* Tratar exceções
* Fechar conexões corretamente

---

## 🔗 Integração com Aplicações Web

* Dados de formulários → banco
* Consultas → exibição na interface
* APIs → comunicação com frontend

---

## ✅ Resumo

* Acesso é feito via linguagem + driver
* CRUD é base das operações
* Segurança e performance são essenciais
